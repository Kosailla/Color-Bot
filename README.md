**Description**

   This Color Bot is an automation tool that monitors the screen and performs actions based on detecting specific colors.


    
    Target Zone: 2x2 pixels - The specific area being monitored

    Check Delay: 0.009 seconds - How frequently the bot checks for the color

    Toggle Key: F6 - Hotkey to enable/disable the bot

    Customizable: All parameters can be modified including target color and hotkey
